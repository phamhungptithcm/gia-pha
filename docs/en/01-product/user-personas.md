# User Personas

_Last reviewed: March 17, 2026_

## Clan Owner / Clan Admin

Responsibilities:
- define clan structure and governance direction
- approve sensitive changes and oversee data quality
- monitor operations, including billing and membership health

Needs:
- clear authority boundaries
- visible operational status
- trustworthy audit trails

## Branch Admin

Responsibilities:
- maintain member data and relationships in branch scope
- support onboarding and role transitions

Needs:
- branch-safe permissions
- fast member search and reliable forms

## Member

Responsibilities:
- view genealogy context and family connections
- keep own profile up to date
- follow events and updates

Needs:
- readable mobile-first UI
- clear language and low-friction flows

## Parent Proxy

Responsibilities:
- verify child-access login flow
- support minors or members without direct device ownership

Needs:
- safe and simple verification flow
- clear parent-child context separation

## Treasurer / Operations Steward

Responsibilities:
- maintain transparent fund transaction records
- support scholarship workflow traceability

Needs:
- reliable balance visibility
- clear financial history and references

## Release Maintainer (Engineering/Ops/QA)

Responsibilities:
- maintain CI/CD reliability
- ensure production configuration integrity
- enforce release quality gates

Needs:
- predictable pipelines
- fast diagnosis for deploy/runtime issues
