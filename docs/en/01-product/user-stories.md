# User Stories

_Last reviewed: March 17, 2026_

## Identity and Access

- As a family member, I can sign in with phone OTP and reach the correct clan context.
- As a child account user, I can access with child identifier and parent verification.
- As a pre-created member, I can claim my profile safely to link account and role.

## Clan and Member Operations

- As a clan owner/admin, I can manage clan and branch structure.
- As a branch admin, I can manage members in my allowed scope.
- As a member, I can update my profile and avatar.

## Genealogy and Relationships

- As an admin, I can create parent-child and spouse relationships with validation.
- As a member, I can read lineage context in an interactive tree.

## Events and Notifications

- As a clan participant, I can follow solar and lunar events in one workspace.
- As a member, I can receive reminders and read updates from inbox.

## Funds and Scholarship

- As an operator, I can create funds and record transactions with transparent balance impact.
- As a governance reviewer, I can process scholarship submissions in a structured flow.

## Billing and Subscription

- As a clan owner/admin, I can view active plan, expiry, and eligibility to upgrade or renew.
- As a payer, I can complete VNPay checkout via a clear step-by-step flow.
- As a payer, I can understand payment outcome states: success, pending, failed/canceled.
- As a clan, upgraded entitlement is granted only after confirmed successful payment.
